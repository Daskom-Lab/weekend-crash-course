Reverse Engineering
Binary Exploitation (pwn)
Mengandalkan kesalahan programmer dalam memory handling dan sebagainya
CTF Challenges :
  • Buffer overflow
  • Off by one byte
  • Heap Exploitation
Web Exploitation (pwn)
Mengandalkan kesalahan programmer dalam banyak hal
CTF Challenges :
  • XSS (Cross Site Scripting)
  • SQLi (SQL injection)
  • CSRF (Cross Site Request Forgery)
  • SSRF (Server Side Request Forgery)
  • LFI (Local File Inclusion)
  • RCE (Remote Code Execution)
  • TOC-TOU (Time of Check Time of Use)
Forensic Analysis
Mencari suatu hal yang (biasanya) tersembunyi dalam hal lain
CTF Challenges :
  • Dumped Memory analysis (Pada OS Level)
  • Image Analysis (Steganography)
  • Sound Analysis
OSINT (Open Source Intelligence)
Mencari suatu hal yang ada di internet (Open Sourced), biasanya digunakan untuk mencari orang hilang / mencari aset aset perusahaan (berguna untuk mencari hidden bug juga)
CTF Challenges :
  • Find missing person
  • And many more
